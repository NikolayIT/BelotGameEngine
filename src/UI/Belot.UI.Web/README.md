# Belot.UI.Web

Уеб приложение за игра на белот онлайн с използване на SignalR за комуникация в реално време.

## Възможности

- Създаване на игрални стаи
- Присъединяване към съществуващи стаи
- Поддръжка на до 4 играча в една стая
- Комуникация в реално време чрез SignalR
- Визуална игрална маса с позиции на играчите

## Стартиране

```bash
cd src/UI/Belot.UI.Web
dotnet run
```

Приложението ще стартира на адрес: http://localhost:5000

## Технологии

- ASP.NET Core 8.0
- SignalR за real-time комуникация
- HTML/CSS/JavaScript за front-end
- Belot.Engine за логиката на играта

## Използване

1. Отворете браузър на адрес http://localhost:5000
2. Въведете вашето име и име на стаята
3. Натиснете "Създай стая" или изберете съществуваща стая от списъка
4. Изчакайте други играчи да се присъединят (необходими са 4 играча)
5. Когато всички 4 играчи са в стаята, играта започва автоматично

## Структура

- **Models/** - Модели за игрални стаи и играчи
- **Services/** - Бизнес логика за управление на стаи и играчи
- **Hubs/** - SignalR хъб за комуникация
- **wwwroot/** - Статични файлове (HTML, CSS, JavaScript)
